PROJECT 01

밀리의 서재

Open API, Responsive Website

point color

국내 도서 사이트인 "밀리의 서재"의 클론 사이트입니다.

페이지 내의 도서 이미지 및 제목은 Kakao에서 제공하는 Open Api 데이터를 받아 사용하였습니다.
미디어를 이용하여 다양한 화면에서의 대응할 수 있도록 하였고, jquery를 사용하여 스크롤과 클릭에 반응하도록 작업하였습니다.

jquery의 사용에 있어서는 비슷한 결과물이라도 반복문을 이용하여 클릭에 대응해보는 식으로 여러방식의 접근을 시도 해보았습니다.
